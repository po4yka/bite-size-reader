## Backend Selection and Rollback

`MIGRATION_INTERFACE_BACKEND` controls interface-layer routing behavior:

- `rust` (required): interface routing path is Rust-only.
- Legacy values (`canary`, `python`) are decommissioned for runtime execution.

Related controls:

- `MIGRATION_INTERFACE_TIMEOUT_MS` (default `150`)
- `INTERFACE_ROUTER_RUST_BIN` (optional binary path override)

Python fallback for this slice is decommissioned.

## CI Coverage

- New CI job: `Migration Interface Router Parity (M4)` in `.github/workflows/ci.yml`.
- Verifies fixture baselines are up to date and runs Rust M4 parity tests.
