## Runtime Defaults and Decommission

M5 defaults production traffic to Rust-first paths:

- `SUMMARY_CONTRACT_BACKEND=rust`
- `MIGRATION_INTERFACE_BACKEND=rust`

For migrated slices (M2 contract layer, M4 interface routing), Python fallback behavior is decommissioned.

## Runtime Failure Visibility

Cutover monitor events are written to `MIGRATION_CUTOVER_EVENTS_FILE` when Rust paths fail.
These events are for operational visibility and incident response; they do not trigger Python fallback.
