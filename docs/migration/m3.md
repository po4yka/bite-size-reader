## Shadow Mode Configuration

`MIGRATION_SHADOW_MODE_ENABLED` controls whether M3 shadow comparisons run.

- `false` (default): disabled (no comparison path)
- `true`: run Python-authoritative flow and compare snapshots against Rust shadow path

Additional controls:

- `MIGRATION_SHADOW_MODE_SAMPLE_RATE` (default `0.0`)  
  - Sampling ratio in `[0.0, 1.0]`
- `MIGRATION_SHADOW_MODE_TIMEOUT_MS` (default `250`)  
  - Per-slice Rust invocation timeout
- `MIGRATION_SHADOW_MODE_MAX_DIFFS` (default `8`)  
  - Max diff paths emitted per mismatch log
- `MIGRATION_SHADOW_MODE_EMIT_MATCH_LOGS` (default `false`)  
  - Emit explicit match logs in addition to mismatch/error logs
- `PIPELINE_SHADOW_RUST_BIN` (optional)  
  - Explicit binary path override for `bsr-pipeline-shadow`
