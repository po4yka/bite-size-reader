# Chroma Schema Reference

Source: `app/infrastructure/vector/chroma_schemas.py`

## ChromaMetadata Fields

All fields validated by Pydantic on upsert. `extra="forbid"` rejects unknown keys.

| Field | Type | Required | Constraints | Description |
|---|---|---|---|---|
| `request_id` | `int` | yes | `>= 0` | Ratatoskr request correlation ID |
| `summary_id` | `int` | yes | `>= 0` | Summary record ID |
| `user_id` | `int \| None` | no | `>= 1` | Telegram user ID (multi-tenant) |
| `user_scope` | `str` | yes | alphanumeric + `-_`, lowercased | Tenant scope (injected by store) |
| `environment` | `str` | yes | alphanumeric + `-_`, lowercased | Environment namespace (injected by store) |
| `text` | `str` | yes | 1-20000 chars, stripped | Embedding source text |
| `language` | `str \| None` | no | -- | Language code (en, ru, auto) |
| `url` | `str \| None` | no | -- | Source article URL |
| `title` | `str \| None` | no | -- | Article title |
| `source` | `str \| None` | no | -- | Content source identifier |
| `published_at` | `str \| None` | no | -- | Publication date string |
| `window_id` | `str \| None` | no | -- | Chunk window identifier |
| `window_index` | `int \| None` | no | `>= 0` | Chunk window position |
| `chunk_id` | `str \| None` | no | -- | Semantic chunk identifier |
| `neighbor_chunk_ids` | `list[str]` | no | max 128, deduped | Adjacent chunk references |
| `created_at` | `str \| None` | no | -- | Embedding creation timestamp |
| `tags` | `list[str]` | no | max 128, deduped | Topic tags |
| `topics` | `list[str]` | no | max 128, deduped | Topic taxonomy labels |
| `semantic_boosters` | `list[str]` | no | max 128, deduped | Semantic boost terms |
| `local_keywords` | `list[str]` | no | max 128, deduped | Chunk-local keywords |
| `local_summary` | `str \| None` | no | -- | Chunk-local summary text |
| `query_expansion_keywords` | `list[str]` | no | max 128, deduped | Query expansion terms |
| `section` | `str \| None` | no | -- | Article section label |

**List field validation:** All list fields are cleaned (deduped, stripped, max 128 items). Accepts `str`, `list`, `set`, `tuple`, or `None`.

## ChromaQueryFilters

Used to build Chroma `where` clauses. Source: `ChromaQueryFilters.to_where()`.

| Field | Type | Required | Description |
|---|---|---|---|
| `environment` | `str` | yes | Injected by store |
| `user_scope` | `str` | yes | Injected by store |
| `language` | `str \| None` | no | Language filter |
| `tags` | `list[str]` | no | Tag containment filters (max 64) |
| `request_id` | `int \| None` | no | Filter by request ID |
| `summary_id` | `int \| None` | no | Filter by summary ID |
| `user_id` | `int \| None` | no | Filter by user ID |

### `to_where()` Logic

- **Without tags:** Returns flat dict `{"environment": ..., "user_scope": ..., [optional filters]}`
- **With tags:** Returns `{"$and": [...conditions...]}` where each tag becomes `{"tags": {"$contains": tag}}`

## ID Generation

IDs are generated by `ChromaVectorStore._extract_id()`:

```
Priority:
1. request_id + chunk_id  -> "123:chunk_abc"
2. request_id + window_id -> "123:window_0"
3. request_id + summary_id -> "123:456"
4. request_id alone       -> "123"
5. Fallback               -> uuid4().hex
```

## Collection Settings

- **Naming:** `notes_{environment}_{user_scope}_{version}` (spaces replaced with `_`)
- **Distance:** HNSW with cosine space (`"hnsw:space": "cosine"`)
- **Similarity:** `similarity = 1.0 - cosine_distance` (clamped to [0.0, 1.0])
- **Metadata on collection:** `environment`, `user_scope`, `version`

## Embedding Models

| Language | Model | Dimensions | Notes |
|---|---|---|---|
| `en` | `all-MiniLM-L6-v2` | 384 | English-optimized |
| `ru` / `auto` | `paraphrase-multilingual-MiniLM-L12-v2` | 384 | Multilingual |
| (Gemini) | `gemini-embedding-2-preview` | 768 | Cloud provider, configurable dims (1-3072) |

Model selection: `EmbeddingService._get_model_name_for_language()` checks registry, falls back to default multilingual model. Gemini provider available via `EMBEDDING_PROVIDER=gemini`.
